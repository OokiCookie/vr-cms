Экспорт и Импорт данных
📊

Экспорт в Excel (.xlsx)

Экспортирует все записи в формате Excel — как оригинальный «Обученные VR по площадкам», с листами по месяцам.

📋

Сводка «Факт обучения»

Сводная таблица по предприятиям × курсам × месяцам — формат третьего документа «Факт обучения VR».

💾

Резервная копия (JSON)

Сохраняет всю базу данных в JSON-файл. Используйте для резервного копирования или переноса данных между компьютерами.

📂

Импорт из JSON

Загрузите ранее сохранённый JSON-файл. Данные будут добавлены к текущим (дубликаты исключаются по ID).

📑

Импорт из Excel

Загрузите Excel-файл в формате «Результаты обучения» (первый документ). Данные будут добавлены к текущей базе.

🗑

Очистить базу

Удаляет все записи из текущей базы данных. Перед очисткой автоматически создаётся бэкап в облаке.

💾 Бэкапы базы данных
Хранится последние ${''} бэкапов. Автоматический бэкап создаётся раз в неделю при первой загрузке приложения.

Анкета обратной связи

QR-код будет сгенерирован автоматически из этой ссылки.

Создать группу курсов

Группа объединяет несколько курсов справочника визуально. Каждый курс сохраняет свой QR и счётчик записей; группа показывает общую сумму.

Отметьте курсы, которые войдут в эту группу.

⚠ Очистить базу данных?

Будут удалены все записей из базы. Это действие отображается во всех отчётах.
